RACES Training Operations

Open Database File

Opens and loads a couple of database tables (Member and Status) into the application. Displays the size of the tables.

Open Spreadsheet CSV File

Opens and loads the CSV file downloaded previously from the Google Training Spreadsheet. Displays the contents of the spreadsheet. Scroll down to see it all.

Compare the two data sets

Only the differences are displayed. See Introduction for description of differences.

Print

Will do all the normal things to print a file.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.
